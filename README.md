## How to push work onto GitHub
 
 
1.     Save work of Pycharm 
2.     Open terminal on Pycharm
3.     First command (git init)
4.     Followed by git status (optional)
5.     git add . 
6.     git commit -m “update”
7.     git push -u origin main
